# Code of Conduct

This project adopts the Contributor Covenant Code of Conduct.

We are committed to fostering an open, inclusive, and respectful community for everyone.

Participants are expected to:

* Be respectful and constructive.
* Welcome different perspectives and experiences.
* Focus on collaboration and learning.
* Provide helpful and professional feedback.

Unacceptable behavior includes harassment, discrimination, personal attacks, or any conduct that creates an unwelcoming environment.

Project maintainers reserve the right to moderate discussions and contributions to ensure a positive experience for all participants.

For questions regarding the Code of Conduct, please open a private discussion with the project maintainers.
